Variables

Para crear una variable con alcance global, declárela dentro del :root selector. El :rootselector coincide con el elemento raíz del documento.

Usamos la var()función para insertar el valor de las variables.

Tradicional

body { background-color: #1e90ff; }

h2 { border-bottom: 2px solid #1e90ff;

Alternativa

:root {

--blue: #1e90ff;

--white: #ffffff;

}
